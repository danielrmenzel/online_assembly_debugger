# TinyCC Build Instructions

To rebuild the TinyCC WebAssembly (WASM) modules used in this project, follow these steps:


Build the TinyCC WASM module:
   ```bash
   ./build_tcc_wasm.sh
   ```

---

# Other Instructions

For building additional WebAssembly modules (e.g., Capstone, Unicorn), please refer to the official documentation provided by their respective maintainers.

> ⚠️ **Note:** No modifications were made to the files generated by the original builds.
